认识 Spring Data JPA

Java Persistence API

JPA 为对象关系映射提供了⼀种基于 POJO 的持久化模型

  • 简化数据持久化代码的开发⼯作
  • 为 Java 社区屏蔽不同持久化 API 的差异

Spring Data

在保留底层存储特性的同时，提供相对⼀致的、基于 Spring 的编程模型

主要模块

  • Spring Data Commons
  • Spring Data JDBC
  • Spring Data JPA
  • Spring Data Redis

定义 JPA 实体对象

常⽤ JPA 注解

实体

  • @Entity 注明这个类是一个实体
  • @MappedSuperclass
  • @Table(name) 实体和表关联

主键

  • @Id
    • @GeneratedValue(strategy, generator)
    • @SequenceGenerator(name, sequenceName)

映射

  • @Column(name, nullable, length, insertable, updatable)
  • @JoinTable(name)、@JoinColumn(name)

关系

  • @OneToOne、@OneToMany、@ManyToOne、@ManyToMany
  • @OrderBy

通过 Spring Data JPA 操作数据库

Repository

@EnableJpaRepositories 注解，自动发现 Repository 的扩展

Repository 接⼝

  • CrudRepository<T,ID>
  • PagingAndSortingRepository<T,ID>
  • JpaRepository<T,ID>

定义查询

根据⽅法名定义查询

  • find…By… / read…By… / query…By… / get…By…
  • count…By…
  • …OrderBy…[Asc / Desc]
  • And / Or / IgnoreCase
  • Top / First / Distinct

分⻚查询

  • PagingAndSortingRepository <T, ID>
  • Pageable / Sort
  • Slice / Page<T, ID>

Repository 是怎么从接⼝变成 Bean 的

Repository Bean 是如何创建的

JpaRepositoriesRegistrar

  • 激活了 @EnableJpaRepositories
  • 返回了 JpaRepositoryConfigExtension

RepositoryBeanDefinitionRegistrarSupport.registerBeanDefinitions

  • 注册 Repository Bean（类型是 JpaRepositoryFactoryBean ）

RepositoryConfigurationDelegate.registerRepositoriesIn

  • 注册找到的 repositories

RepositoryConfigurationExtensionSupport.getRepositoryConfigurations

  • 取得 Repository 配置

JpaRepositoryFactory.getTargetRepository

  • 创建了 Repository

接⼝中的⽅法是如何被解释的

RepositoryFactorySupport.getRepository 添加了Advice

  • DefaultMethodInvokingMethodInterceptor
  • QueryExecutorMethodInterceptor

AbstractJpaQuery.execute 执⾏具体的查询

语法解析在 Part/PartTree 中

通过 MyBatis 操作数据库

简单配置

  • mybatis.mapper-locations = classpath*:mapper/**/*.xml
  • mybatis.type-aliases-package = 类型别名的包名
  • mybatis.type-handlers-package = TypeHandler扫描包名（类型转换处理器）
  • mybatis.configuration.map-underscore-to-camel-case = true

Mapper 的定义与扫描

  • @MapperScan 配置扫描位置
  • @Mapper 定义接⼝
